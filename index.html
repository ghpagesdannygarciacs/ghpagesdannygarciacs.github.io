Some Personal Projects I've Built

JP Visual

JP Visual is a program that translates Japanese from any given image. The program uses an object detection algorithm to locate potential letters and translate them using Google Translate's python API. Knowing the input language cuts down on the potential mis-identifications and improves performance over similar more generic algorithms. This is a work in progress and can currently only identify Hiragana. I'm working on implementing Katakana and Kanjis, as well as an app that uses the algorithm.

Python, Temp, Temp

Boid Flock Emergent AI

Algorithm used to simulate the movement of birds, fish, and other natural flocking creatures through an emergent system (system that does not depend on its individual parts, but on their relationships to one another). The algorithm is written to work on a 2D environment and is usable on any context that satisfies that condition, such as this very website. For details on the mechanics, visit the extended GitHub version or read Craig Reynolds' paper called "Flocks, Herds, and Schools: A Distributed Behavioral Model" on which this work is based.

Technologies Used:

Javascript, HTML, CSS

Market Screener is a program that fetches and displays real-time market data along with a multitude of customizable analysis metrics.
